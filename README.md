# User API - JUnit Testing Example

Projeto desenvolvido para demonstrar testes automatizados em Java usando **JUnit**, **Spring Boot** e **GitHub Actions**.

## Tecnologias utilizadas

- Java 17
- Spring Boot
- Maven
- JUnit 5
- Mockito
- H2 Database
- GitHub Actions
